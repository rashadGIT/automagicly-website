INSTRUCTIONS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Copy GOOGLE_PRIVATE_KEY from your .env file
2. Copy SUPABASE_ANON_KEY and SUPABASE_SERVICE_ROLE_KEY from your .env file
3. Choose a strong NEXT_PUBLIC_ADMIN_PASSWORD
4. In AWS Amplify Console, add each variable one by one:
   - Key: Variable name (e.g., GOOGLE_SERVICE_ACCOUNT_EMAIL)
   - Value: The actual value

⚠️  SECURITY WARNING:
   - Never commit .env files to Git (they're already in .gitignore)
   - Keep SUPABASE_SERVICE_ROLE_KEY secret
   - Keep GOOGLE_PRIVATE_KEY secret
   - Use a strong ADMIN_PASSWORD

After adding all variables, Amplify will automatically redeploy your app.
